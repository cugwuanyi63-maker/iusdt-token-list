IUSDT – capped, fee-aware USDT-pegged token

IUSDT is a fixed-decimal ERC-20 token on zkSync Era designed to follow the value of Tether USD (USDT). It integrates directly with Uniswap v3 and uses a capped supply model, fee-on-transfer mechanics, and optional staking rewards.

The token is deployed with 6 decimals and a hard cap of 100,000,000,000 IUSDT. Liquidity is seeded in a Uniswap v3 pool against native USDT at a 1:1 starting price, allowing users to swap between IUSDT and USDT on zkSync Era.

How it works

Swapping & usage

IUSDT is intended to track USDT value on zkSync Era through a direct Uniswap v3 pool. Users can:

  • Swap between IUSDT and USDT using Uniswap v3 on zkSync Era.
  • Hold IUSDT as a USDT-pegged asset within the zkSync ecosystem.
  • Use IUSDT in other DeFi applications that integrate ERC-20 tokens on zkSync Era.

To swap, open Uniswap, switch the network to zkSync Era, then paste the IUSDT contract address: 0xf927397a7c603073DF19098b8c115d40acc0dB15.
